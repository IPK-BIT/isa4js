# Security Policy

## Supported Versions
Security updates are actively provided for the following versions of isa4js:
| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |

## Reporting a Vulnerability
If you discover a security vulnerability within `isa4js`, please do **not** open a public issue.

Instead, please report it using GitHub's **Private Vulnerability Reporting** tool:

1. Navigate to the **Security** tab of this repository.
2. Click on **Advisories** in the left-hand sidebar.
3. Click the **Report a vulnerability** button.
4. Provide a clear descriptive summary and the steps to reproduce the vulnerability
